# Initialization Sequence (Detailed)

## Day 0

1. Establish ownership model (single feature owner by default).
2. Define high-risk gate policy and hard blockers.
3. Create initial `AGENTS.md`/`CLAUDE.md` project guidance.
4. Add core templates/checklists.

## Day 1-7

1. Execute first feature through full loop.
2. Capture first compound artifact.
3. Validate triage workflow and backlog hygiene.

## Day 8-30

1. Start weekly compounding ritual.
2. Populate maturity scorecards for critical domains.
3. Close highest-friction process gaps.
