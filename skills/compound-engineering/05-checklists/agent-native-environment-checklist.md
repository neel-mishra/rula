# Agent-Native Environment Checklist

## Development Access
- [ ] Agent can read/write project files
- [ ] Agent can run tests and lint/type checks
- [ ] Agent can inspect local logs
- [ ] Agent can run local app(s)

## Git and Delivery Access
- [ ] Agent can create branches and commits
- [ ] Agent can open PRs
- [ ] Agent can read review feedback

## Runtime and Debugging Access
- [ ] Agent can capture UI evidence (screenshots where needed)
- [ ] Agent can inspect network/error traces where needed
- [ ] Agent has bounded access to operational diagnostics

## Safety and Governance
- [ ] Risky operations require explicit documented controls
- [ ] Rollback mechanisms are documented and testable
- [ ] Audit-relevant artifacts are retained
